# Golden Star FC ⚽

🔗 **Demo en vivo:** https://luisfernando1930.github.io/landing-club-fc/

Landing page de un club de fútbol amateur ficticio — historia, plantilla de jugadores y próximos partidos.

## Tecnologías usadas

- HTML5 semántico
- CSS3 (variables, Flexbox, Grid, media queries)
- Diseño responsive (mobile-first)
- Git y GitHub (flujo con branches y Pull Requests)

## Características

- Navegación por anclas entre secciones
- Diseño adaptable a celular, tablet y escritorio
- Tarjetas de jugadores y partidos organizadas en grid
- Sección de contacto con enlaces funcionales (correo y teléfono)

## Cómo ejecutarlo localmente

1. Clona este repositorio
2. Abre `index.html` con Live Server (o cualquier servidor local)
